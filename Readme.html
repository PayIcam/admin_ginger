Projet Web du Gala des Icam 2012

Le but de ce site internet est de proposer différents outils concernant les inscriptions, les entrées et le vestiaire du Gala.

Les inscriptions :
Le site permettra de rentrer les inscriptions au Gala dans une base de données commune. Cela facilitera le recoupage des informations après chaque séance d'inscriptions.
Les entrées :
Le jour même du Gala, le site permetra de s'occuper des entrées pour vérifier si la personne qui se présente a bien été invité. Cela permettra aussi une évolution en temps réel du nombre total de personnes présentes à la soirée.
Les vestiaires :
Le jour même du Gala, le site permetra de s'occuper des vestiaires. En exploitant la même base de données que le site des entrées, on pourra consulter la progression des invitations et l'on pourra associer le vestiaire de la personne qui le souhaite à son nom.
